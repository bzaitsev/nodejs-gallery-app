## Description
In this app user can upload images on server and all images are displayed in the gallery.

**Tech stack**: Node.js, jQuery, Webpack

## Run project locally
To launch server, in root folder run:  
```sh
npm i
npm run nodemon
```
Open in browser http://localhost:3000
